Installation
============

You can install the package as follows:

.. tab-set::

    .. tab-item:: latest version

        Using pip

        .. code-block:: console

            (.venv) $ pip install git+https://github.com/mipt-srf/probe-station

        or uv

        .. code-block:: console

            $ uv add git+https://github.com/mipt-srf/probe-station

    .. tab-item:: latest PyPI release

        Using pip

        .. code-block:: console

            (.venv) $ pip install probe-station

        or uv

        .. code-block:: console

            $ uv add probe-station
